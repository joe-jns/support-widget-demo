# Support Widget Demo

Widget de support flottant — UX concierge premium avec avatar animé, articles populaires et chat IA simulé.

**Live demo :** https://joe-jns.github.io/support-widget-demo/

## Utilisation

Ouvre `index.html` dans n'importe quel navigateur. Zéro dépendance, pas de build, pas de serveur nécessaire.

## Fonctionnalités

- Bouton flottant rond avec animation pulse
- Deux tabs : **Aide** (articles + recherche) et **Chat** (assistante IA Maya)
- Avatar Maya animé (clignement des yeux, respiration)
- Bulle de greeting qui apparaît automatiquement après 1,8s
- Greeting selon l'heure (Bonjour / Bon après-midi / Bonsoir)
- 3 articles populaires avec vue article inline
- Feedback article (👍/👎) — "Non" redirige vers le chat
- Barre de recherche (Enter → ouvre le chat avec la question)
- Chat IA simulé avec matching mots-clés (workflow, calendrier, email, snapshot)
- Chips contact direct Email & WhatsApp

## Tech

- HTML / CSS / JavaScript vanilla
- Fichier unique (`index.html`), zéro dépendance
- Thème clair, accent `#F9A637`
- Border-radius max 10px
- Responsive mobile

## Note

C'est une démo visuelle. Le chat est simulé (réponses hardcodées par mots-clés), les liens de contact utilisent des adresses fictives.
